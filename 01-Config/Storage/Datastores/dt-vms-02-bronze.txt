Name: dt-vms-02-bronze

01 - Connect to the ESXi host with SSH.

02 - Stop the USB arbitrator service. This service is used to passthrough USB device from an ESX/ESXi host to a virtual machine. (When disabling it, you can no longer passthrough USB devices to VMs).
  /etc/init.d/usbarbitrator stop

03 - Use this command to permanently disable the USB arbitrator service after reboot.
  chkconfig usbarbitrator off

04 - Plug in the USB device to your ESXi host.

05 - List the disks.
  ls /dev/disks/

06 - Write a GPT label to the device.
  partedUtil mklabel /dev/disks/mpx.vmhba34:C0:T0:L0 gpt

07 - To create a partition you need to know the start sector, end sector, which depends on the device size and the GUID.
  The start sector is always 2048
  The GUID for VMFS is AA31E02A400F11DB9590000C2911D1B8
  The end sector can be calculated with the following formula (Use the numbers from getptbl):

  partedUtil getptbl /dev/disks/mpx.vmhba34:C0:T0:L0
    gpt
    121597 255 63 1953458176

  eval expr $(partedUtil getptbl /dev/disks/mpx.vmhba34:C0:T0:L0 | tail -1 | awk '{print $1 " \\* " $2 " \\* " $3}') - 1
    1953455804

  partedUtil setptbl /dev/disks/mpx.vmhba34:C0:T0:L0 gpt "1 2048 1953455804 AA31E02A400F11DB9590000C2911D1B8 0"

08 - Create the VMFS partition.
  ** Replace with your end sector

  vmkfstools -C vmfs6 -S dt-vms-02-bronze /dev/disks/mpx.vmhba34:C0:T0:L0:1
